# PDFJS Sample

https://github.com/mozilla/pdf.js

Note: Experimental repository for pdfjs

## Getting Started

```sh
$ git clone git@github.com:terut/pdfjs_sample.git
$ cd pdfjs_sample

# ruby
$ ruby -run -e httpd . -p 8080

# python > 3.x
$ python -m http.server 8080

# php > 5.4.0
$ php -S localhost:8080
```
